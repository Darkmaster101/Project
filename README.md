                                                  Weather Bot
**Weather Bot** is a Python desktop application with a modern graphical interface designed to retrieve and display the 
current weather in a user-selected city.
The program uses the OpenWeather API to retrieve up-to-date weather data from the internet. The user enters the name of 
the city, which the application displays:

- current temperature
- perceived temperature
- description of weather conditions
- air humidity
- wind speed
- weather icons

The project was developed using the **Tkinter** library and includes a user-friendly interface with animations and 
color schemes
Moreover, the application supports:
- Input error handling
- Internet connection testing
- Storing launch history by using the Django API
- Displaying user session history


                                                   Technologies used

When developing this bot, many technologies were used, such as:
- Python
- Tkinter
- Requests
- Django API
- OpenWeather API
- Threading


                                               Installation and Launch instructions

To run this bot correctly, you need to do a few things:
1) Install Python
2) Download the project
3) Install the libraries from requirements.txt (`pip install -r requirements.txt`)
4) Start the Django server (`python run.py`)
